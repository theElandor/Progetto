Descrizione dettagliata

Classe che gestisce il salvataggio automatico.

Classe che estende il Graphics.DialogHandler, in modo da ereditare i metodi per comunicare messaggi all'utente attraverso la finestra di log.

Metoto specifico che gestisce il salvataggio automatico.

Prima di chiamare il metodo update_save, si controlla se il file è un file di backup generato dal metodo stesso. In tal caso, il salvataggio automatico non viene effettuato.
