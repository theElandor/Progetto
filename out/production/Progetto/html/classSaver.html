boolean SaveLoad.Saver.update_save ( boolean  Backup)
inline

Se il file non e' ancora stato salvato allora viene chiamato il metodo "save()".

Altrimenti viene sovrascritto il vecchio salvataggio, come in excel. Backup viene passato come "true" se il metodo e' chiamato dall'autosaver, altrimenti e' false. In questo modo uso lo stesso metodo per due casistiche diverse, usando un parametro di supporto. Se Backup e' false, allora la funzione sovrascrive il salvataggio precedente. Se Backup e' true, allora la funzione crea un nuovo file dal nome "model.getCurrentSave()+~". La funzione update_save viene chiamata automaticamente da un thread ogni 30 secondi con il parametro Backup = true, in modo da simulare un salvataggio automatico, utile in caso di crash.
