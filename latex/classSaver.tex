Se il file non e\textquotesingle{} ancora stato salvato allora viene chiamato il metodo \char`\"{}save()\char`\"{}. 

Altrimenti viene sovrascritto il vecchio salvataggio, come in excel. Backup viene passato come \char`\"{}true\char`\"{} se il metodo e\textquotesingle{} chiamato dall\textquotesingle{}autosaver, altrimenti e\textquotesingle{} false. In questo modo uso lo stesso metodo per due casistiche diverse, usando un parametro di supporto. Se Backup e\textquotesingle{} false, allora la funzione sovrascrive il salvataggio precedente. Se Backup e\textquotesingle{} true, allora la funzione crea un nuovo file dal nome \char`\"{}model.\+get\+Current\+Save()+$\sim$\char`\"{}. La funzione update\+\_\+save viene chiamata automaticamente da un thread ogni 30 secondi con il parametro Backup = true, in modo da simulare un salvataggio automatico, utile in caso di crash.
