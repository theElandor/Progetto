Metodo che svolge la funzionalita\textquotesingle{} principale della classe. 

~\newline
 Ritorna false se esce con errore, se invece il caricamento avviene correttamente ritorna true.~\newline
 Fa uso di un J\+File\+Chooser per far scegliere all\textquotesingle{}utente il file da caricare, poi esegue i controlli sull\textquotesingle{}estensione.~\newline
 Sono consentiti soltanto file con estensione .ser o -\/ser$\sim$.~\newline
 Legge da file le due strutture dati fondamentali, cioè la tabella delle celle e la \mbox{\hyperlink{classValueTable}{Value\+Table}} contenente i valori numerici.~\newline
 Se la lettura avviene con successo, le due strutture dati vengono inserite all\textquotesingle{}interno del Table\+Model.~\newline
 Vengono poi salvate due informazioni\+: il path del salvataggio corrispondente alla corrente istanza della tabella, e un booleano che indica l\textquotesingle{}esistenza di questo salvataggio, per semplicità. In tal modo quando il \mbox{\hyperlink{classSaver}{Saver}} dovrà effettuare delle operazioni di salvataggio, dovrà soltanto leggere quella stringa.~\newline
 Inoltre l\textquotesingle{}\mbox{\hyperlink{classAutoSaver}{Auto\+Saver}} effettuerà un salvataggio di backup soltanto se l\textquotesingle{}attuale istanza della tabella è già stata salvata.(Può facilmente ottenere questa informazione valutando se il booleano è True / False).~\newline
